# Swift-Clean
# 🧼 DryCleaning App

## 📌 Project Overview
The **DryCleaning App** is an online dry-cleaning service that allows customers to:
- Select items they want to clean or iron.
- Get assigned to the nearest dry-cleaning shop.
- Track order progress and delivery.

Dry cleaners also get a dedicated system to manage customer orders efficiently.

## 🚀 Features
### **For Customers:**
✅ **Easy Order Placement** – Select clothes, choose services, and place an order quickly.  
✅ **Real-Time Tracking** – Follow order status updates.  
✅ **Secure Payments** – Multiple payment options.  
✅ **User-Friendly Interface** – Simple navigation and intuitive design.  

### **For Dry Cleaners:**
✅ **Order Management** – View and process customer requests.  
✅ **Delivery Scheduling** – Organize pickup and drop-off logistics.  
✅ **Business Dashboard** – Insights into earnings and orders.

## 📖 Installation & Setup
1. **Clone the Repository:**  
   ```bash
   git clone https://github.com/your-username/drycleaning-app.git
   cd drycleaning-app
   ```
2. **Install Dependencies:**  
   ```bash
   flutter pub get
   ```
3. **Run the App:**  
   ```bash
   flutter run
   ```
